## 🧑‍💻 About Me

Software Engineer with **~5 years** of experience building scalable web applications and data-driven platforms. I specialize in **backend development** with **Node.js** and **AWS**, with a strong focus on designing **event-driven architectures** powered by **RabbitMQ**, **Redis**, and **BigQuery**.

- 🔭 Most recently built a **real-time user behavior analytics platform** (Google Analytics–style) that lets marketers capture, analyze, and optimize user engagement across websites.
- 🤖 Shipped **AI-Intent** and **AI-Insight** — LLM-powered features for record-level enrichment and recurring report analysis, backed by multi-provider LLMs, BigQuery, and AWS Lambda/SQS/DynamoDB.
- 🧩 I care about the unglamorous parts — throughput, idempotency, and observability — that make products feel instant and never go down.
- 🌱 Currently going deeper into **distributed systems** and looking for the next hard problem at scale.
- 📍 Based in Nha Trang, Vietnam · open to select opportunities.

> *"Good architecture isn't the cleverest code — it's the system that keeps working at 3 AM."*
